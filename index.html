A cidade define a posição exata do ponto no mapa (base IBGE + coordenadas GeoNames).
Para salvar de forma permanente, o site grava o arquivo empresas.json no seu repositório via API do GitHub. Sem token, as alterações ficam só neste navegador.
Como criar o token: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → só o repositório do mapa, permissão Contents: Read and write.
O token fica salvo apenas no seu navegador (localStorage). Nunca comite o token no repositório.
